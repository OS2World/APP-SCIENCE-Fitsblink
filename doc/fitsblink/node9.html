8. Erasing an image

If you don't want to use the currently selected image (its name is displayed in the File box) anymore, just press the Erase button and the image will be removed from the memory. Of course, the file on disk is not affected. If this image is also displayed at the moment when you press Erase, it will be replaced by the next image in the file list or with an empty screen if this is the only file currently loaded.
